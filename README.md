# Azure Hybrid Cloud Security Lab

This project demonstrates building a hybrid cloud infrastructure and securing it using Microsoft Azure.

Project Goals:
- Build Azure networking infrastructure
- Connect on-premise network to Azure
- Deploy secure servers
- Implement firewall and segmentation
- Implement identity security
- Detect threats using Microsoft Sentinel

Technologies Used:
- Microsoft Azure
- Azure Virtual Network
- Azure Firewall
- Azure Bastion
- Azure AD
- Microsoft Sentinel
